# Changelog

All notable changes to the AI-Me Vendor Card Specification will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/), and this project adheres to [Semantic Versioning](https://semver.org/) for the specification version.

## [0.1] - 2026-03-05

### Added

- Initial draft specification (`spec/ai-me-v0.1.md`)
- JSON Schema for validation (`schemas/ai-me.schema.json`)
- Four example vendor cards: SaaS, consulting, open-source, e-commerce
- CLI validator tool (`packages/validator/`)
- Documentation website at [spec.ai-me.dev](https://spec.ai-me.dev)
- Interactive validator playground
- Schema.org mapping documentation
- Community files: CONTRIBUTING, CODE_OF_CONDUCT, GOVERNANCE, SECURITY
- CI pipeline (GitHub Actions)
